# Acknowledgements

NISB is an independent project, but it stands on the shoulders of the open-source community.

## Open-Source Ecosystem

Special thanks to the maintainers and contributors of the open-source projects used by NISB, including projects in the Python, JavaScript, Vue, document processing, AI tooling, and self-hosted infrastructure ecosystems.

## Core Technology Areas

NISB benefits from open-source work in many areas:

- Web application frameworks.
- Python API services.
- Document parsing and conversion.
- PDF and Office document processing.
- Vector search and data processing.
- Browser-based editing.
- Markdown rendering and sanitization.
- Visualization.
- Self-hosted deployment infrastructure.
- AI SDKs and model integration tools.

## Standards and Protocols

NISB also benefits from open standards and protocol work around web applications, document formats, containerized deployment, API specifications, and AI tool integration.

## Users and Testers

Thank you to early users, testers, and readers who help improve NISB through feedback, bug reports, suggestions, and real-world usage.
